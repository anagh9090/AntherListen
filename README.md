# AntherListen

**AntherListen** is a open-source project, forked from **Music Player Go**.
Visit **Music Player Go** here:         [Music Player Go Github](https://github.com/enricocid/Music-Player-GO)
### About
**AntherListen** is made because **I** don't like any other open-source local music player, so i made one of my own, forking from **Music Player Go**, which I liked because it was *KISS*. This is for only **Android**.
